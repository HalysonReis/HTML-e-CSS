Testando carga de imagens

Para carregar imagens no site usa-se a tag img

Nesta tag tem dois parametros o src= e o alt=

O src= é o caminho da foto

O alt= é o texto alternativo uma descrição da foto

Para fazer o colar o caminho um atalho é apertar control + espaço
